Minecraft — это 3D-«песочница», разработанная компанией Mojang Studios, где игрок взаимодействует с игровым миром через размещение и разрушение различных блоков в трёх разных измерениях.


Есть два основных изданий майнкрафта:

Minecraft: Java edition версия майнкрафта для пк, на ней поддерживается большое количество пользовательский модов. Она менее оптимизирована чем второе издание и написано на языке java.

Minecraft: bedrock edition это второе издание которое вышло для большинства платформ: windows, linux, android, ios, xbox и playstation.
